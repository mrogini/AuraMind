# AuraMind

AuraMind is a Mental Wellness and Mood Tracking Platform developed using Java Full Stack technologies.

## Features

* User Registration and Login
* Daily Mood Tracking
* Weekly Mood Reports
* Monthly Mood Reports
* Personality Assessment Test
* Mood History Management
* Admin Dashboard
* Question and Option Management
* Overall Analytics and Reports

## Technology Stack

### Frontend

* HTML
* CSS
* JavaScript
* JSP

### Backend

* Java
* Servlets

### Database

* MySQL

### Build Tool

* Maven

### Server

* Apache Tomcat
